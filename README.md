# Currency Converter (C Console)

Консольное приложение для конвертации валют, написанное на чистом C. Оптимизировано для работы на Debian 12 (VPS).

## Файлы
- **valute.c**: Основной программный код. Реализует HTTP-запросы через `libcurl` и логику поиска валют.
- **cJSON.c / cJSON.h**: Легковесная библиотека для парсинга JSON (необходимо компилировать вместе с основным файлом).
- **valute**: Исполняемый бинарный файл.

## Зависимости
Для сборки и работы необходимы:
- Компилятор `gcc`
- Системная библиотека `libcurl4-openssl-dev`

Установка зависимостей на Debian/Ubuntu:
```bash
apt update && apt install -y gcc libcurl4-openssl-dev
```

## Как собрать и запустить
В языке C важно указывать все используемые `.c` файлы при компиляции.

1. **Скомпилировать:**
   ```bash
   gcc valute.c cJSON.c -o valute -lcurl
   ```
2. **Запустить:**
   ```bash
   ./valute
   ```

## Особенности реализации
- **Управление памятью**: Программа использует динамическое выделение памяти для обработки ответов от API. Вся память очищается вручную (`free`) для предотвращения утечек.
- **Интеллектуальный поиск**: При вводе неверного кода (например, `USX`), программа анализирует базу данных API и предлагает существующие варианты (например, `USD`, `USN`).
- **Регистр**: Программа автоматически переводит ввод пользователя в верхний регистр (`usd` -> `USD`).

## API
Используется бесплатный ключ **ExchangeRate-API**. Данные о курсах обновляются в реальном времени.
