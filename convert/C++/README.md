# Currency Converter (C++ Console)

Консольное приложение для работы на Linux.

## Файлы
- **valute.cpp**: Основной исходный код. Использует `libcurl` для сетевых запросов.
- **json.hpp**: Библиотека для парсинга JSON ответов от API.
- **valute**: Исполняемый бинарный файл (появляется после компиляции).

## Зависимости
Для работы требуются системные библиотеки:
- `libcurl4-openssl-dev`

## Как запустить
1. Скомпилировать: `g++ valute.cpp -o valute -lcurl`
2. Запустить: `./valute`

## Логика подсказок
Если ввести несуществующий код (например, `RUX`), программа предложит варианты на ту же букву (`RUB`, `RWF` и т.д.), используя данные напрямую из API.
