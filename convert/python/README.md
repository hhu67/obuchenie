Описание создано для работы на Linux

Скачайте python3:
```bash
sudo apt update && sudo apt install python3 -y
```
Или:
```bash
sudo pacman -Syu python
```
Установите модуль requests:

```bash
sudo apt install python3-requests
```
Запустите файл, находясь в одном катологе с файлом:
```bash
python3 valute.py
```
Или:
```bash
uv run --with requests valute.py
```
